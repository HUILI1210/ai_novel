# 👑 白金蔷薇：温柔公主的骑士誓约

## 剧本信息
- **剧本ID**: preset_princess
- **角色名称**: 艾琳娜 (Elena)
- **故事类型**: 奇幻宫廷恋爱
- **字数规模**: 完整长篇剧情

## 角色设定

### 艾琳娜公主
- **性格**: 温柔治愈、善良体贴、脆弱敏感、在逆境中觉醒成长
- **外貌**: 金色长卷发如流金、宝石蓝眼眸、完美S型曲线、白色镶金礼服、纯白高透丝袜
- **身份**: 艾尔兰王国长公主，后加冕为女王

### 玩家（骑士）
- **性格**: 忠诚、高大、沉默可靠
- **身份**: 皇家骑士，公主专属护卫

## 故事背景
中世纪欧洲风格奇幻王国，魔法与剑的时代

---

## 场景结构（7个场景）

### 【场景 01】序章：蔷薇庭院的初遇
- **地点**: 皇家花园（正午，阳光斑驳）
- **BGM**: Morning Sunlight - 轻柔明亮的钢琴曲
- **CG 01**: 花园长椅上，艾琳娜捧书阅读，阳光洒在金发上
- **剧情**: 骑士初次觐见公主，两人相遇，建立信任契约

### 【场景 02】第一章：图书馆的微光与体态
- **地点**: 皇家图书馆（午后，静谧）
- **BGM**: Afternoon Tea - 悠扬的大提琴
- **CG 02**: 公主踮脚取书，裙摆微提特写
- **剧情**: 日常守护，骑士扶住险些摔倒的公主，关系升温

### 【场景 03】第二章：破碎的白丝与共犯
- **地点**: 废弃花园（阴天，萧瑟）
- **BGM**: Wounded Heart - 略带忧伤的风笛
- **CG 03**: 公主被荆棘划伤，骑士跪地包扎
- **剧情**: 公主躲避逼婚，受伤后骑士悉心照料，情感深化

### 【场景 04】第三章：晚宴的拥抱与宣示
- **地点**: 宴会厅（夜晚，灯火辉煌）
- **BGM**: Royal Waltz - 华丽的圆舞曲
- **CG 04**: 骑士用披风护住公主，对峙公爵
- **剧情**: 建国日晚宴，骑士保护公主免受公爵骚扰，公开宣示守护

### 【场景 05】第四章：雨夜的誓言
- **地点**: 公主寝宫（暴雨夜，雷声）
- **BGM**: Rain and Tears - 悲伤激昂的钢琴曲
- **CG 05**: 公主赤脚穿白丝，泪眼婆娑
- **剧情**: 政变前夜，公主向骑士倾诉恐惧，骑士立下永恒誓言

### 【场景 06】第五章：血色黎明与女王觉醒
- **地点**: 王宫城墙（黎明，战火硝烟）
- **BGM**: Battle of Roses - 激昂的管弦乐
- **CG**: 无（战斗场景用背景+立绘）
- **剧情**: 叛军攻城，骑士负伤，公主觉醒持剑护卫，蜕变为女王

### 【场景 07】终章：加冕之夜（True End）
- **地点**: 女王寝宫（深夜，烛光）
- **BGM**: Eternal Vow - 浪漫舒缓的弦乐
- **CG 06**: 女王转身，月光下的身影
- **CG 07**: 两人十指紧扣特写
- **剧情**: 加冕典礼后，两人终成眷属，达成True End
